## Dev

Iniciando: 
```npm install```

> Todo o código-fonte está no diretório sources. Não altere os arquivos fora dele.

Rodando o task runner (modo desenvolvimento):
```grunt dev```

Rodando o task runner (modo produção):
```grunt```

## Dependências

- task runner: Grunt
